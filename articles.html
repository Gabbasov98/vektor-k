Насос для бассейна: ключевой элемент для поддержания качества воды
одяной насос для бассейна — это устройство, которое используется для циркуляции воды в искусственном водоеме. Принцип его работы довольно прост. Он засасывает воду из бассейна, прогоняет ее через фильтр и возвращает обратно в бассейн. Фильтр задерживает грязь, мусор и бактерии, делая воду чистой. Исходя из назначения насоса, выделяют два вида насосных агрегатов: с предфильтром и без предфильтра. Первый используется в фильтрационных системах. Второй же обеспечивает постоянный поток жидкости для водных аттракционов: водопадов, противотечений и других.
